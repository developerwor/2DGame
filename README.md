## 🤖 블록 피하기 게임

- 2D 환경 프로젝트

## 📂 주요 코드 기능

```bash
< 프로세스 간략한 설명 >
   |
   |-- Block_fall.cs/                
   |    |-- 장애물 블럭 낙하/
   |
   |-- Char_Move.cs/                           
   |    |-- 캐릭터 이동/                    
   |
   |    |-- 에리어 이탈시 복귀/                 
   |
   |-- Main_flowcam.cs/                           
   |    |-- 카메라 동기화/                 
   |         |-- 캐릭터의 x축만 카메라 동기화
   |-- ************************************************************************
```


```bash
< 프로세스 세부적인 설명 >
   |
   |-- Block_fall.cs/                
   |    |-- 블럭 아래 이동/
   |         |-- 블럭 스피드 타입 랜덤지정/          
   |         |-- 블럭 에리어 이탈시/          
   |             |-- 블럭 복귀 위치 지정/
   |             |-- 블럭 스피드타입 랜덤지정/
   |             |-- 블럭 복귀 위치 x값 랜덤지정/     
   |
   |-- Char_Move.cs/                           
   |    |-- 오른쪽 키 입력/                 
   |         |-- 캐릭터 오른쪽 이동                
   |         |-- 걷는 모션 실행
   |         |-- 캐릭터 오른쪽으로 방향전환
   |         
   |    |-- 왼쪽 키 입력/                 
   |         |-- 캐릭터 왼쪽 이동                
   |         |-- 걷는 모션 실행
   |         |-- 캐릭터 왼쪽으로 방향전환
   |         
   |    |-- 스페이스 키 입력/                 
   |         |-- 캐릭터 점프                
   |         |-- 날아가는 모션 실행
   |
   |    |-- 에리어 바깥 이동/                 
   |         |-- 캐릭터 복귀 위치로 이동
   |
   |-- Main_flowcam.cs/                           
   |    |-- 오른쪽 키 입력/                 
   |         |-- 캐릭터가 X축으로 이동한만큼 카메라 오른쪽 이동
   |
   |    |-- 왼쪽 키 입력/                 
   |         |-- 캐릭터가 -X축으로 이동한만큼 카메라 왼쪽 이동
   |-- ************************************************************************
```

<br />

## 📚 사용된 라이브러리

- `UnityEngine` : 유니티의 기본적인 라이브러리 게임 오브젝트, 컴포넌트, 물리 시뮬레이션 등을 다루는 기능
